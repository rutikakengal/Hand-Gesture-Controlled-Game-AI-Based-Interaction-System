## 🌟 Overview

An AI-powered computer vision project that allows users to control browser-based games like **Subway Surfers 🚇** and **Temple Run 🐒** using real-time hand gestures via webcam.

## ✨ Features

- 🎯 Real-time hand gesture recognition  
- 🎮 Control game movements (Left, Right, Jump, Slide)  
- 🌐 Works with browser-based games  
- ⚡ Lightweight and fast execution  
- 🧠 AI-based hand tracking

---

## 🎮 Controls

| Gesture | Action |
|--------|--------|
| 👍 Thumb Up | Move Left |
| Pinky Finger Up | Move Right |
| ✋ All Fingers Up | Jump |
| ✊ Fist (No Fingers) | Slide |

---

## 🛠️ Tech Stack
| Technology | Purpose |
|--------|-------------|
| Python |	Core Programming |
| OpenCV |	Image Processing |
| CVZone |	Hand Tracking Module |
| MediaPipe	| Hand Landmark Detection |
| PyAutoGUI	| Keyboard Automation |

---

## 🧠 How It Works

```text
Webcam → Hand Detection → Finger Recognition → Gesture Mapping → Keyboard Input → Game Control
```

---

## 📌 Usage
- Open a browser game (Subway Surfers / Temple Run)
- Click on the game window
- Run the script
- Control the game using gestures

---

## ⚠️ Requirements
- Webcam
- Good lighting conditions
- Active game window
